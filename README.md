# 📚 AI Learning Tracker App

An AI-powered, cross-platform learning tracker built with:
- 🧠 Spring Boot (Microservices)
- ⚡ Kafka & Redis
- 🐦 Flutter (GetX / Dio)
- 📦 Docker + Kubernetes
- 🤖 OpenAI integration

## 📁 Project Structure

- `backend/`: Spring Boot microservices
- `frontend/`: Flutter mobile app
- `docs/`: Diagrams, notes, architecture
